## Deployment

This site is deployed via **Netlify** with drag-and-drop or GitHub auto-deploy.

- `netlify.toml` — build settings and cache headers
- `_redirects` — clean URLs (e.g. `/catering` → `catering.html`)
- Forms use **Netlify Forms** (no backend needed)
